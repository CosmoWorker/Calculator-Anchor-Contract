## Calculator Program
This is a simple calculator contract made using anchor framework for writing contracts and tests for the solana blockchain. 

### Key Features 
* Double
* Add
* Half
* Subtract
* Key tests for program
## Dependencies
* Yarn
* Anchor CLI
* Rustup

## Setup
* ```cd /Calculator-Anchor-Contract```
* ```yarn install```
* ```anchor build```
* ```anchor test```
